# CommonsBase_Remote

`CommonsBase_Remote` hosts dk remote-execution UI rules that are consumed by
`dk0 remote ...` from other repositories.

The current package target is:

- `CommonsBase_Remote.GitHub.Run@0.1.0`

This repository is structured after `CommonsBase_FileMagic` and is the package
home for the GitHub remote orchestration rule that was previously prototyped
in-tree.
